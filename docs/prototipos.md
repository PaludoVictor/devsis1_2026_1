# 6. Protótipos de Telas

Esta seção apresenta a materialização visual do **Sistema de Gestão Pet Shop Amigo Fiel**, demonstrando a experiência de uso (UX) e os fluxos de interface (UI) planejados para os operadores do sistema.

Apresentamos abaixo as interfaces de usuário (UI) projetadas para a operação diária do Pet Shop:

![Tela de Login](/entrar.jpeg)

*Interface de autenticação para controle de acesso seguro dos operadores ao sistema.*

### 💻 2. Tela de Recepção

![Tela de Ficha Clínica](/ficha_clinica.jpeg)

### 📝 6. Tela de Prontuário
![Tela de Prontuário](/prontuario.jpeg)

### 📜 7. Tela de Histórico de Consultas

### 🛒 8. Tela de Frente de Caixa (PDV)
